# Wish Summer Products Sales Dashboard

Інтерактивний аналітичний дашборд Power BI для аналізу продажів літніх товарів інтернет-магазину **Wish**.

## 📋 Про проект

Курсова робота з дисципліни **«Аналітика великих даних»**  
Спеціальність: 122 «Комп’ютерні науки»  
Національний університет харчових технологій (НУХТ), 2024

**Тема:** Аналіз даних статистики продажів літніх товарів інтернет-магазину Wish (2020–2021 роки)

## 🎯 Основні можливості дашборду

- Ключові показники ефективності (KPI): загальний прибуток, кількість продажів
- Порівняння результатів за 2020 та 2021 роки
- Аналіз сезонної динаміки продажів (по місяцях та кварталах)
- Географічний розподіл виробників товарів (інтерактивна карта)
- Аналіз рейтингів товарів (розподіл оцінок від 1 до 5 зірок)
- Детальна інформація по товарах і продавцях
- Інтерактивне **дерево декомпозицій** (drill-down аналіз прибутку)

## 🛠 Технології та інструменти

- **Power BI Desktop**
- **Power Query** — очищення та трансформація даних
- **DAX** — розрахункові міри (Total Revenue, Profit 2020, Profit 2021 тощо)
- Модель даних: Main_Table, Product_Details_Table, Sellers_Table, Calendar

## 📊 Сторінки дашборду

1. **Загальні відомості** — загальна картина продажів і KPI
2. **Деталі** — аналіз товарів, рейтингів і географії
3. **Дерево декомпозицій** — багаторівневий аналіз прибутку

## 📁 Структура репозиторію

- `reports/` — файл дашборду `.pbix`
- `screenshots/` — скріншоти сторінок дашборду
- `data/` — зразки даних (опціонально)
- `docs/` — курсова робота (Word)
